👋 Bienvenue !

  1. ➕ Commencez par aller dans la section Catégories pour organiser vos produits. Cliquer sur le bouton "+ Ajouter" pour créer une catégorie

  7. N’oubliez pas de vérifier les alertes de stock faible sur la page d’accueil
